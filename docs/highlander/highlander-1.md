# Highlander Format Rules

The following supplements the [General Ruleset](/docs/general/general-1). In the event of a conflict, these format-specific rules take precedence.

## Team Composition

9 players per team, one of each class:

| Class | Count |
|-------|-------|
| Scout | 1 |
| Soldier | 1 |
| Pyro | 1 |
| Demoman | 1 |
| Heavy | 1 |
| Engineer | 1 |
| Medic | 1 |
| Sniper | 1 |
| Spy | 1 |

## Class Locking

Each team must field exactly one of each class. No class may be played by more than one player at a time.

## Ringers

A maximum of **2 ringers per map** is permitted in Highlander, and no more than 2 ringers may be active simultaneously. Ringers must comply with all eligibility requirements outlined in the General Ruleset (Section 8).

## Map Format

Payload, Attack/Defense, and KOTH maps will be used in Highlander depending on the season map pool.

Round win conditions follow standard TF2 map objectives.
