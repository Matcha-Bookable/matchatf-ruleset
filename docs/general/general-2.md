# Game Integrity

## Section 3: Game Integrity

### 3.1 Cheating

Cheating in any form is strictly prohibited, including in official matches, scrimmages, PUGs, and public servers. Evidence from demos, server logs, or anti-cheat software will be investigated.

**Penalties:**
- Cheating / Hacking (Aimbot, ESP, etc.): 3-year ban (1st offence), permanent ban (2nd offence)
- VAC / Game ban in TF2: 3-year ban (unless confirmed non-TF2 or removed by Valve)

All matches a banned player participated in during the season will be overturned. Teams with a cheater in playoffs will be disqualified.

Players who knowingly field cheaters in their team rosters may also face penalties.

### 3.2 Third-Party Software

All third-party software is assessed on whether it provides an unfair advantage over other players.

- Audio equalizers used to isolate specific sounds (e.g. spy decloaks) are considered cheating and are banned.
- Mouse software macros for automated turning or movement are banned. A sensitivity toggle is allowed.

If unsure about a specific piece of software, contact an admin before using it in a match.

### 3.3 Scripts

Scripts that allow players to perform actions they could not otherwise perform, or that disrupt the player's hitbox, are banned.

**Banned scripts:**
- Bunnyhop scripts
- Panic scripts
- Charge turn scripts / turn scripts (180° etc.)
- Any script to glitch or freeze a player model/hitbox

**Allowed scripts:**
- Crouch jump scripts
- Null-cancelling movement scripts
- Rocket jump scripts
- Mask uber scripts
- Team say scripts
- Loadout switching scripts
- Viewmodel toggle scripts
- Zoom scripts
- Crosshair changing scripts
- No explosions / particles scripts
- Speedometer scripts

Contact an admin if you are unsure about a specific script.

### 3.4 Game Exploits

Abusing unintended game mechanics to gain an unfair advantage is prohibited. Exploits marked as "cheating ban" below are treated as cheating (see Section 3.1 penalties).

**Prohibited:**
- Bypassing class limits imposed by server config
- Civilian pose
- Sniper scope transparency exploit
- Box shadows (cheating ban)
- Spy cloak / footstep / uber sound scripts or audio mods (cheating ban)
- Transparent props or removed map geometry
- Retry command abuse for faster respawn
- Uber pause bug (both teams must agree to resolve fairly)
- Hitbox desync exploits (Reverse Quantum Crouch, teleport hitbox manipulation)
- Bypassing sv_cheats & sv_pure (cheating ban)
- Killing players during setup time
- Using `record;stop` to detect cloaked spies

**Allowed mods:**
- Flat textures
- Custom HUDs, crosshairs, hitsounds
- No hats / cosmetics mods
- No ambient sounds
- No explosions / particles
- Transparent viewmodels
- Removing damage particles

**Not allowed mods:**
- Modified weapon tracer effects
- Oversized particle effects
- Transparent enemy pyro flames
- External executables modifying textures, models, or particles
- No shadow modification
- Malicious weapon sound replacements that provide a detection advantage

**Allowed bugs:**
- Wallbugs
- Edgebugs
- C-taps (both manual and scripts)
- Crouch sniper peeking / corner sniping

### 3.5 Map and Weapon Exploits

Exploiting map geometry or weapon mechanics to gain an unintended advantage is prohibited. This includes:

- Accessing out-of-bounds areas
- Shooting players from positions where they cannot return fire
- Capping or denying points through walls, floors, or ceilings
- Placing engineer buildings in unreachable or out-of-bounds locations
- Resupplying through improperly brushed spawn zones
- Extending the Pyro flamethrower range through external means
- Infinite airblast glitch, lagging server with phlog, pyroboots (jetpack into resupply for reduced fall damage)
- Taunt killing through walls
- Extending phlogistinator invulnerability

The above list is not exhaustive. Contact an admin if you are unsure about a specific interaction.

### 3.7 Ghosting

Sending or receiving real-time match information to or from someone outside the match (including via spectator mode) constitutes ghosting and will result in suspension for all involved.

### 3.8 Streaming Conduct

Stream sniping is the responsibility of the streamer. The league will not take action against stream snipers; players stream at their own risk.

Recommended practice: delay the stream by 90+ seconds, mute audio or use an overlay during pauses.

### 3.9 Live Coaching

Coaching during an active match is not permitted.

### 3.10 Account Sharing and Alternate Accounts

Each player may have one league account. Multiple accounts or shared accounts are not permitted.

Siblings or housemates sharing an IP must notify an admin in advance and use separate Steam and league accounts.

Penalties: match overturns, major penalties, or league bans depending on severity.
Playing on an account belonging to a permanently banned player: Permanent ban.

### 3.11 Improper Server Administration

Using server operator privileges (such as host or admin) to influence the outcome of an official match is prohibited.
